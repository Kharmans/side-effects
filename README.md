# Side Effects

An interface-focused module providing an effect directory and other small interface changes for D&D 5th Edition. The UI settings in v13 of CPR have been ported to this module. All features are off by default except for the effect interface. For support, please join the [Coven's discord server](https://discord.gg/BumxBcQDrT).

## Features:
- The *Effect Interface*, an effect directory accessible from the sidebar, built from a user-specified compendium.
- Edit status effects and create custom ones in the sidebar.
- *Chat message dark mode* and chat tweaks.
- Hide compendiums from rendering in the compendium directory.
- Add temporary status effects to the token HUD.
- Remove unwanted status effects from the token HUD.
- Preset custom status effect icons.
- Disable blinded and invisible conditions from applying visual effects.
- Item directory context option to export items without a description, for easy sharing.
- Scale the control and navigation buttons.
